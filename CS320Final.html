def check_missing(inp):
  i = 0
  values = df[inp].value_counts()
  if values.get('Unknown') is not None:
    i += values['Unknown']
  if values.get('Missing') is not None:
    i += values['Missing']
  return i

def grapher(inputs, title):
  counts_sym = []
  counts_dth = []
  for item in inputs:
    counts_sym.append((df[df[title] == item]['symptom_status']).value_counts())
    counts_dth.append((df[df[title] == item]['death_yn']).value_counts())
  fig = plt.figure()
  ax = fig.add_axes([0,0,1,1])

  inputs = [i[0:8] for i in inputs]
  ax.bar(inputs, [i['Symptomatic']/i['Asymptomatic'] for i in counts_sym])
  plt.title("Symptomatic/Asymptomatic by " + title)
  plt.show()

  fig = plt.figure()
  ax = fig.add_axes([0,0,1,1])
  ax.bar(inputs, [check_yes(i)/(i['No'] + check_yes(i)) for i in counts_dth])
  plt.title("Death by " + title)
  plt.show()

Conclusions¶

The results of the data analysis show a few different results. The 0-18 age group has the highest amount of asymptomatic cases versus symptomatic cases. The 65+ age group had the highest death rate by far. The American Indian/Alaska Native and Multiple/Other races had the lowest rates of death. The Native Hawaiian/Other Pacific Islander had zero deaths which seems to be an error considering the average for other races. The only difference between hispanic and non-hispanic was the death rate which was a considerable difference. Women were more likely to be symptomatic than men however men were more likely to die from Covid. By far the biggest correlation between a variable and likelyhood of death was underlying conditions. The ratio of symptomatic to asymptomatic was suprisingly low for those with underlying conditions.

The machine learning analysis I did used age group, race, ethnicity, underlying conditions, and gender to predict a yes or no value for death. The model has a weighted accuracy of 0.90 however the accuracy for deaths is very low. This could be due to the lower amount of data as well as the unpredictability of death due to covid. The results of this anaylsis show that being older than 65 and having underlying health conditions are the main factors in contributing to dying from covid.
